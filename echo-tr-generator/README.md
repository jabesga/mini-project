# ECHO | TR GENERATOR

Little python script to generate an 'encrypted' word that can be decrypted with the 'tr' command

Inspired by the email address of [vterron](https://github.com/vterron)

Example:

======== ECHO | TR GENERATOR ========
Insert a phrase or your email:
        JONANDER
Output:
        echo qKNJNH1y | tr 1JKyHqN EAORDJN
========================
